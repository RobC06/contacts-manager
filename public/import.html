📋 How to Import from Monday.com

1

Export from Monday.com

  • Open your Monday.com board with contacts
  • Click the three dots menu (⋯) in the top right corner
  • Select "Export board to Excel"
  • Download the Excel file to your computer
2

Convert to CSV (if needed)

  • Open the downloaded Excel file
  • Go to File → Save As
  • Choose "CSV (Comma delimited) (*.csv)" as the file type
  • Save the file

Note: You can also upload the Excel file directly!

3

Upload Your File

Use the file uploader below to import your contacts. We support:

  • CSV files (.csv)
  • Excel files (.xlsx, .xls)

📊 Required CSV Format

Your file should have these column headers:

Name, Company, Title, Tag, Follow-up Date

Column Mapping: We'll automatically detect columns like "Contact Name", "Full Name", "Company Name", "Organization", "Job Title", "Status", etc. The importer is flexible!

Example:

Name Company Title Tag Follow-up Date
John Doe Acme Corp CEO follow up 2026-02-15
Jane Smith Tech Inc CTO waiting for response 2026-03-01
